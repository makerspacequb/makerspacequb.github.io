Completed Projects

  • RetroPi mini arcade: A mini arcade cabinet powered by a raspberry pi and RetroPie OS.
  • Arduino/ Leap motion library: Sample code for reading serial effectively on arduino and using a leap motion. Was used to create controllable arm (see video in respository)
  • JavaFX virtual browser: JavaFx virtual browser with external control to calling js functions. Ideal for any interactive demo projects using a browser such as interactive maps using external sensors
  • Xbox controller: Source code and libraries for xbox controller integration into java
